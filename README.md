# 🚀 DigiTools - Digital Tools Marketplace

DigiTools is a modern web application where users can explore and purchase premium digital tools like AI writing tools, design templates, and more.

This project is built using React, Vite, and Tailwind CSS.

---

##  Features

- 🛍️ Browse digital products
- ➕ Add products to cart
- 🛒 View cart items
- 🔢 Increase / decrease quantity
- ❌ Remove items from cart
- 💰 Dynamic total price calculation
- 📱 Responsive design (Mobile + Desktop)
- 🎨 Clean and modern UI
- 🔔 Toast notifications (optional)

---

## Technologies Used

- React JS
- Vite
- Tailwind CSS
- DaisyUI
- Lucide React Icons
